Class Ephem

A class representing Kepler ephemerides.

  a: 3.009622263428050E+01,
  e: 7.362571187193770E-03,
  i: 1.774569249829094E+00,
  om: 1.318695882492132E+02,
  w: 2.586226409499831E+02,
