Class SpaceObject

An object that can be added to a visualization.

example
const myObject = viz.addObject('planet1', {
  position: [0, 0, 0],
  scale: [1, 1, 1],
  particleSize: 5,
  labelText: 'My object',
  labelUrl: 'http://...',
  hideOrbit: false,
  ephem: new Spacekit.Ephem({...}),
  textureUrl: '/path/to/spriteTexture.png',
  basePath: '/base',
  ecliptic: {
    lineColor: 0xCCCCCC,
    displayLines: false,
  },
  theme: {
    color: 0xFFFFFF,
    orbitColor: 0x888888,
  },
});
